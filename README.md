## E - Pizza

**É necessário ter o Node.JS LTS em seu computador para execultar o projeto**

Rodando o Projeto

```bash
npm run build
# or
yarn build

and

npm run dev
# or
yarn dev
```
Projeto criado com Next.js para criar um catálogo de pizza.
